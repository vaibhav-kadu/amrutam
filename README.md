# Amrutam Technology

Amrutam is an Android application developed by Amrutam Technology.

## Features
- **WebView Integration**: Seamlessly browse the Amrutam website within the app.
- **Download Management**: Built-in support for downloading files with notifications.
- **Connectivity Monitoring**: Real-time detection of internet status to ensure a smooth user experience.
- **Push Notifications**: Integrated with Firebase Cloud Messaging for timely updates.
- **Latest Android Support**: Optimized for Android 15 (API 35).

## Tech Stack
- **Language**: Java
- **SDK**: Android 15 (Target API 35)
- **Firebase**: Database, Messaging, Crashlytics, Analytics
- **UI Components**: SwipeRefreshLayout, Material Design

## History
This repository contains the development history of the Amrutam application, with backdated commits reflecting the creation of key components starting from April 2024.
